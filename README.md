# CMS data modeling and analysis
The Centers for Medicare & Medicaid Services synthetic public use files include beneficiaries from 2008-20010 and their associated claims files. The purpose of this project is to clean and transform these files to create a simple data mart that allows a more streamlined investigation of medicare population disparities by chronic disease. The end goal is to connect this database to powerbi and create a dashboard similar to the [CMS MMD population view tool](https://data.cms.gov/tools/mapping-medicare-disparities-by-population)
## Data:
- [CMS DE-SynPUF](https://www.cms.gov/data-research/statistics-trends-and-reports/medicare-claims-synthetic-public-use-files/cms-2008-2010-data-entrepreneurs-synthetic-public-use-file-de-synpuf/de10-sample-1) 
- Beneficiary Summary: Demographic information of Beneficiaries
- Carrier Claims
- Inpatient Claims
- Outpatient Claims
